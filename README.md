# GameOfLife

This is one of my school project, a Java version of Conway's Game of Life.

これは私の学校の課題の一つで、コンウェイのライフゲームのJava版です。


Credit to the original inventor: Cambridge mathematician John Conway.

原作者であるケンブリッジ大学の数学者、ジョン・コンウェイに感謝します。

## Run

```powershell
javac *.java
java GameOfLife
```

## Buttons

- Start / Stop
- Step
- Random
- Clear

## Files

- `GameOfLife.java` starts the program
- `GameWindow.java` creates the main window
- `GameBoard.java` displays the cell table
- `GameControls.java` handles the buttons
- `LifeGrid.java` stores alive/dead cells
- `LifeRules.java` calculates the next generation
- `GameStyle.java` stores colors, size, and speed
